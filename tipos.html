Tipos de datos basicos que existen en TypeScript

El sistema de tipos de Typescript realiza una formalización de los tipos de Javascript, mediante una representación estática de sus tipos dinámicos. Esto permite a los desarrolladores definir variables y funciones tipadas sin perder la esencia de Javascript. Poder definir los tipos durante el tiempo de diseño nos ayuda a evitar errores en tiempo de ejecución, como podría ser pasar el tipo de variable incorrecto a una función. JavaScript tiene siete tipos de datos diferentes: Null, Indefinido, Booleano, Número, Cadena, Símbolo (introducido en ES6) y Objeto. TypeScript define algunos tipos más, y todos ellos se tratarán en detalle en este tutorial.

Los tipos basicos son:

Number
Tipo de dato que se usa para representar tanto enteros como valores de punto flotante
String
Tipo de dato que se utiliza para almacenar información textual.
Boolean
Tipo de dato lógico que representa verdadero o falso.
Array
Tipo de dato estructurado que permite almacenar una colección de elementos.
Tuple
Tipo similar al array, pero con un número fijo de elementos escritos.
Enum
Tipo enumeración.
Any
Tipo que indica que la variable puede ser de cualquier tipo. Es muy útil a la hora de trabajar con librerías externas.
Void
Tipo que indica que una función no devolverá ningún valor
Never
Tipo de valores que nunca se producen.
